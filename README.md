## 介紹

此專案為貪吃蛇遊戲，練習 OOP 和 TypeScript

### 使用指南
1. 請先確認本地環境已安裝 Node.js 與 npm 。
2. 將專案 clone 至本地，至終端機輸入：
   ```
   git clone https://github.com/Gino-Hsu/snake-typescript.git
   ```
3. 透過終端機進入資料夾位置，輸入：
   ```
   npm install
   ```
4. 安裝完成後，輸入：
   ```
   npm start
   ```
5. 停止使用，至終端機：
   ```
   ctrl + c
   ```
